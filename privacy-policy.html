Privacy Policy

Application: N8N Content Automation
Last updated: March 28, 2026

1. Overview

N8N Content Automation ("the App") is a personal automation tool that integrates with the TikTok Content Posting API to schedule and publish video content. This application is operated exclusively by its owner for personal use only. No third parties, clients, or end-users interact with this application.

2. Data We Collect

This application does not collect, store, or process any personal data from any user. Specifically:

  • No personal information is gathered from visitors or third parties.
  • No analytics or tracking tools are used.
  • No cookies are set or read by this application.
  • No databases or data storage systems retain user information.

3. TikTok API Usage

This app authenticates with the TikTok platform using OAuth 2.0 on behalf of the app owner only. The following API scopes are used:

  • video.publish — to publish video posts to the owner's TikTok account
  • video.upload — to upload video files to TikTok servers

Access tokens obtained through OAuth are used solely to interact with the TikTok API and are never shared, logged to external services, or stored beyond the scope of the automation workflow.

4. Data Sharing

We do not sell, trade, or otherwise transfer any data to third parties. No data is shared with advertising networks, analytics providers, or any external services.

5. Data Retention

Since no personal data is collected, there is no data to retain or delete. OAuth tokens are managed by the n8n automation engine and are not persisted in any external storage.

6. Security

The application runs locally or on a self-hosted server accessible only to the owner. All communications with TikTok are conducted over HTTPS.

7. Changes to This Policy

This privacy policy may be updated from time to time. Changes will be reflected by updating the "Last updated" date at the top of this page.

8. Contact

If you have any questions about this privacy policy, you can contact the app owner directly through the TikTok Developer portal or the registered developer email address.
